# Pongorithm [![Build Status](https://travis-ci.org/chmontgomery/Pongorithm.png?branch=master)](https://travis-ci.org/chmontgomery/Pongorithm)

app to track ping pong ranking

## Quickstart

1. install dependencies
    * `npm install`
    * `bower install`
2. build project
    * `grunt build`
3. start server
    * `npm start`
